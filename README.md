# Weather Forecast Email

A Python project that uses the OpenWeather API to check weather conditions and automatically sends an email reminder if rain is expected.

## Features
- Fetches weather forecast using OpenWeather API
- Detects rainy weather conditions
- Sends email notifications using Gmail SMTP
- Uses environment variables to protect secrets

## Technologies Used
- Python
- Requests
- SMTP
- OpenWeather API
- python-dotenv

## Installation

pip install -r requirements.txt

## Run

python main.py
